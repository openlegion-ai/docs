# OpenLegion Docs

Documentation site for [OpenLegion](https://github.com/openlegion-ai/openlegion) — autonomous AI agent fleets, isolated, auditable, and production-ready.

Powered by [Mintlify](https://mintlify.com).

## Development

Install the Mintlify CLI to preview docs locally:

```
npm i -g mint
```

From the docs directory, run:

```
mint dev
```

Preview at `http://localhost:3000`.
